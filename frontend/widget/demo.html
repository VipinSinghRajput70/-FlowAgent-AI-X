Embeddable Chat Widget Demo

This landing page demonstrates the deployable FlowAgent AI widget. You can trigger the floating chat bubble on the bottom right of this screen to query the multi-agent system directly. It communicates in real-time with the local FastAPI backend API.

To embed this chat helper on any web page, add the following script tag before the closing </body> tag:

<script src="http://localhost:8000/widget/chat-widget.js"></script>
